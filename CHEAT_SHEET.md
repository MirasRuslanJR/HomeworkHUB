## 📝 Шпаргалка по действиям

| Что хочу | Как сделать |
|----------|-------------|
| **Создать класс** | Аватар → Управление классом → Создать класс |
| **Войти в класс** | Получить код → Управление классом → Присоединиться |
| **Добавить ДЗ** | "+ Добавить ДЗ" → Заполнить → Отправить |
| **Выполнить ДЗ** | Кликнуть задание → Загрузить фото → Отметить |
| **Посмотреть рейтинг** | Боковое меню → 🏆 Рейтинг |
| **Пожаловаться** | Открыть чужое фото → Нажать 🚩 |
